**This directory consists of functions of malloc, dynamic allocation and free**
- file 0:  function that creates an arry of characters, and initializes it with a specific char.

- file 2: Function that concatenates two strings.
- file 3: Function that returns a pointer to a 2 dimensional array of integers.
